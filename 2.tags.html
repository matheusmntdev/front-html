Lista de Tags HTML

1. Estrutura do Documento

<html> - Define o documento HTML.
<head> - Contém metadados do documento.
<title> - Define o título da página.
<body> - Contém o conteúdo visível da página.

2. Texto e Formatação

<h1> a <h6> - Títulos do maior ao menor.
<p> - Define um parágrafo.
<br> - Quebra de linha.
<hr> - Linha horizontal.
<strong> - Negrito semântico.
<b> - Negrito não semântico.
<em> - Itálico semântico.
<i> - Itálico não semântico.
<mark> - Texto destacado.
<small> - Texto pequeno.
<sub> - Texto subscrito.
<sup> - Texto sobrescrito.
<blockquote> - Citação em bloco.
<q> - Citação curta.
<pre> - Mantém a formatação do texto.
<code> - Define código fonte.

3. Links e Navegação

<a> - Define um link.
<nav> - Define uma seção de navegação.
<button> - Botão interativo.
<link> - Referencia arquivos externos.

4. Listas

<ul> - Lista não ordenada.
<ol> - Lista ordenada.
<li> - Item da lista.
<dl> - Lista de descrição.
<dt> - Termo da descrição.
<dd> - Definição do termo.

5. Tabelas

<table> - Define uma tabela.
<tr> - Linha da tabela.
<th> - Cabeçalho da tabela.
<td> - Célula de tabela.
<caption> - Título da tabela.

6. Mídia (Imagens, Áudio e Vídeo)

<img> - Insere imagens.
<audio> - Insere áudio.
<video> - Insere vídeo.
<figure> - Agrupa mídia com legenda.
<figcaption> - Legenda de mídia.

7. Formulários e Inputs

<form> - Define um formulário.
<input> - Campo de entrada.
<label> - Define um rótulo.
<textarea> - Campo de texto multilinha.
<select> - Menu suspenso.
<option> - Opção dentro do select.
<button> - Botão de envio.

8. Estrutura Semântica

<header> - Cabeçalho.
<footer> - Rodapé.
<main> - Conteúdo principal.
<section> - Seção da página.
<article> - Conteúdo independente.
<aside> - Barra lateral.
<div> - Contêiner genérico.
<span> - Contêiner inline genérico.

9. Elementos de Script e API

<script> - Código JavaScript.
<noscript> - Alternativa para navegadores sem JavaScript.
<canvas> - Área de desenho.
<svg> - Gráficos vetoriais.

10. Metadados e SEO

<head> - Contém metadados.
<meta> - Define metadados (charset, viewport, etc.).
<title> - Define o título da página.
<base> - Define a URL base.
