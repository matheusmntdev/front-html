Apresentação sobre Formulários em HTML

1. O que é um Formulário HTML?

Um formulário em HTML é um mecanismo usado para coletar informações dos usuários. Ele pode ser utilizado para várias finalidades, como realizar cadastros, fazer pesquisas, etc.

Exemplo: Um formulário de cadastro de usuário.

2. A Tag <form>

A tag <form> é a estrutura básica para um formulário. Ela contém os campos de entrada e também os atributos que determinam como e para onde os dados serão enviados.

<form action="URL" method="POST">

Atributos importantes da tag <form>:

  • action: Define para qual URL os dados do formulário serão enviados.
  • method: Define o método HTTP usado para o envio (geralmente POST ou GET).

3. Exemplos de Campos de Entrada

A seguir estão alguns exemplos de campos de entrada que você pode usar dentro de um formulário:

Campo de Texto (nome)

<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome">

Descrição: O campo de texto permite que o usuário insira informações simples, como seu nome.

Campo de Senha (senha)

<label for="senha">Senha:</label>
<input type="password" id="senha" name="senha">

Descrição: O campo de senha oculta o que é digitado para proteger informações confidenciais.

Botão de Envio (submit)

<input type="submit" value="Enviar">

Descrição: O botão de envio envia os dados do formulário para o servidor.

4. Campos de Seleção

Você também pode usar campos de seleção, como caixas de seleção e botões de rádio:

Campo de Gênero (botões de rádio)

<label for="masculino">Masculino</label>
<input type="radio" id="masculino" name="genero" value="masculino">

<label for="feminino">Feminino</label>
<input type="radio" id="feminino" name="genero" value="feminino">

Descrição: Os botões de rádio permitem que o usuário selecione apenas uma opção entre várias.

Campo de Seleção (menu suspenso)

<label for="pais">Selecione seu País:</label>
<select id="pais" name="pais">
    <option value="brasil">Brasil</option>
    <option value="portugal">Portugal</option>
</select>

Descrição: O campo de seleção permite que o usuário escolha uma opção de uma lista suspensa.

5. Validação de Formulários

Você pode adicionar validação diretamente no HTML para garantir que os dados inseridos pelos usuários estejam no formato correto.

  • required: Torna o campo obrigatório.
  • type="email": Garante que o campo contenha um endereço de e-mail válido.
  • type="number": Garante que o campo aceite apenas números.

Exemplo com Validação

<input type="email" id="email" name="email" required>

Descrição: Neste exemplo, o campo de e-mail é obrigatório e só aceita valores em formato de e-mail.

6. Exemplo Completo de Formulário

Abaixo está um exemplo completo de formulário com diversos campos:

<form action="/cadastro" method="POST">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>

    <label for="email">E-mail:</label>
    <input type="email" id="email" name="email" required>

    <label for="senha">Senha:</label>
    <input type="password" id="senha" name="senha" required>

    <label for="genero">Gênero:</label>
    <input type="radio" id="masculino" name="genero" value="masculino"> Masculino
    <input type="radio" id="feminino" name="genero" value="feminino"> Feminino

    <label for="comentarios">Comentários:</label>
    <textarea id="comentarios" name="comentarios"></textarea>

    <input type="submit" value="Enviar">
</form>
