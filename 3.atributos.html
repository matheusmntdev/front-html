Atributos do HTML

Os atributos do HTML são informações adicionais fornecidas às tags para modificar seu comportamento ou aparência. Eles são escritos dentro da tag de abertura e seguem a estrutura: 

        <tag atributo="valor">Conteúdo</tag>

Exemplo de um link:

        <a href="https://www.exemplo.com" target="_blank">Visite o site</a>

1. Atributos Globais

Os atributos globais podem ser aplicados a qualquer elemento HTML.

Atributo Descrição
id Define um identificador único para um elemento.
class Define uma ou mais classes CSS para o elemento.
style Adiciona estilos CSS diretamente ao elemento.
title Texto exibido ao passar o mouse sobre o elemento.

2. Atributos Específicos de Tags

Atributos da Tag <a> (Links)

Os atributos da tag <a> controlam o comportamento dos links.

Atributo Descrição
href Define o destino do link.
target Especifica onde abrir o link (_self, _blank, _parent, _top).

Atributos da Tag <img> (Imagens)

Os atributos da tag <img> definem propriedades da imagem.

Atributo Descrição
src Define o caminho da imagem.
alt Texto alternativo exibido caso a imagem não carregue.

3. Atributos de Eventos (Interação com JavaScript)

Os atributos de eventos são usados para definir ações disparadas por interações do usuário.

Atributo Descrição
onclick Executa uma ação quando o elemento é clicado.
onmouseover Executa uma ação quando o mouse passa sobre o elemento.

4. Atributos Booleanos

Os atributos booleanos são aqueles que não precisam de valores, apenas sua presença já os ativa.

Atributo Descrição
checked Define um checkbox ou radio como pré-selecionado.
disabled Desativa o elemento.

5. Atributos Personalizados (data-*)

Os atributos data-* permitem armazenar dados personalizados.

        <button data-id="123" data-usuario="Matheus">Clique aqui</button>
    
    Javascript
    
      const botao = document.querySelector("button");
      console.log(botao.dataset.id); // 123
      console.log(botao.dataset.usuario); // Matheus

Conclusão

Os atributos do HTML são essenciais para modificar o comportamento dos elementos e interagir com CSS e JavaScript.

  • Globais – Aplicáveis a qualquer tag.
  • Específicos – Apenas para certas tags (ex: href para <a>).
  • Eventos – Para interações com JavaScript (ex: onclick).
  • Booleanos – Ativados apenas com sua presença.
  • Personalizados – Criados com data-* para uso em scripts.
